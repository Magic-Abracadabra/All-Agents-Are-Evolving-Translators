# All Agents Are Evolving Translators
## Magic
This is a platform in favor of:
Grammar Correction, Speech or Remote Control, Constructing Self-Defined Command, Interaction with Other Programming Languages, Computing Numbers, Screenshot, Easy File Management, History Conversation, Streaming Answering, Interactive Interface (collaborated with Lee), Natural Language to Command Code (see our research proposal for more details).

We have defined a new Sequence Similarity Metric to map any text to our command sequences. To learn more, go to https://github.com/Magic-Abracadabra/sequence for more details.

The codes will not be updated until we make them open.
